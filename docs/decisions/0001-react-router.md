# ADR-0001: 商品画面のルーティングに React Router を採用

- ステータス: 採用
- 日付: 2026-07-16

## 背景
商品一覧から `/product/:id` へ遷移し、直接アクセスとブラウザー履歴を扱う必要がある。

## 決定
ルーティングに `react-router-dom` 7 を使用する。

## 理由
React の宣言的な画面構成で URL パラメーターと履歴を標準的に扱え、今後のページ追加にも対応しやすい。

## 却下した代替案
History API の自前実装は依存を増やさない一方、URL 監視、履歴、リンク動作を独自に保守する必要があるため採用しない。

## 影響
ランタイム依存とバンドルサイズが増える。採用版は Node.js 20 以上、React 18 以上と互換性がある。
