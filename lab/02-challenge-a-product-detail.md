# 02. 課題A：商品詳細ページを作る（ローカル Agent Mode 向き）

商品カードをクリックすると詳細ページ（`/product/:id`）に遷移し、
大きな画像・商品名・説明・価格を表示する機能を追加します。

## なぜローカル Agent Mode か

レイアウトや画面遷移は **ブラウザで見ないと正解が分からない** ためです。
`npm run dev` で表示を見ながら、その場で修正する同期的なループが向いています。

> **使い分けの判断ルール**
> **目で見て確かめたい → ローカル Agent Mode → Push**

## 前提

- [01. セットアップ](./01-setup.md) の「ローカル実行（課題A用）」で開発サーバーを起動していること。
- キャンバス / ブラウザで `http://localhost:5173/` を開いて表示を確認できること。

## 進め方（例）

1. GitHub Copilot App の Agent に、商品詳細ページの追加を依頼します。
2. `npm run dev` の画面（キャンバス）で表示を確認します。
3. レイアウトや遷移が意図と違えば、その場で追加指示して修正します。
4. 満足のいく状態になったら変更を Push します。

## 実装の目安

- 商品カードのクリックで `/product/:id` に遷移する。
- 詳細ページに **大きな画像・商品名・説明・価格** を表示する。
- 一覧へ戻る導線を用意する。
